# Data Sources

The core football data is StatsBomb Open Data for Major League Soccer 2023, competition ID `44` and season ID `107`.

Generated local tables:

- `match_manifest.csv`: Match inventory and event coverage.
- `player_match_features.csv`: Player-match feature table generated from event and lineup JSON.

The project previously used generic synthetic tables. Those tables are no longer the analytical source of truth. The current artifact uses public event data plus clearly labeled workflow metadata generated by `scripts/score_operating_data.py`.
