# Data Sources

Synthetic source-style data for football scouting predictive analytics.

- `entities.csv`: 36 player prospect records.
- `daily_metrics.csv`: 5,040 player prospect-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
